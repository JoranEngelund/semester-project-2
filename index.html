How Find It Works

  • You can search on auction items titles
  • You can sort by newest and oldest items
  • You can filter the items by tags
  • You can browse the list of auction items

How Bid It Works

  • You can view specific items listed for auction
  • You can view how many bids the listed item has
  • You can bid a desired amount of credits on the item
  • You can view when the auction ends and when it started

How List It Works

  • You can add listing title and description of the item
  • You can set the when the listing ends with date and hours
  • You can add media gallery with up to 3 images
  • You can add desired tags that corresponds with the item
